Prerrequisitos

Instala .NET SDK (por ejemplo .NET 6 o .NET 7) en tu máquina.

Instala Node.js (versión compatible con Angular en el proyecto).

Instala el CLI de Angular globalmente:

npm install ‐g @angular/cli


Ten instalado un SQL Server (o SQL Express) localmente o accesible, y asegúrate de que puedas crear bases de datos, usuarios, etc.

Ten un editor de código como Visual Studio Code o Visual Studio según tu preferencia.
