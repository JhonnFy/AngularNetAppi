## Prerrequisitos

Antes de comenzar con la instalación y ejecución del proyecto, asegúrate de tener lo siguiente instalado y configurado en tu máquina:

1. **.NET SDK**  
   Instala .NET SDK (por ejemplo, .NET 6 o .NET 7) desde la página oficial:  
   [https://dotnet.microsoft.com/download](https://dotnet.microsoft.com/download)

2. **Node.js**  
   Instala Node.js (versión compatible con Angular en el proyecto).  
   [https://nodejs.org/](https://nodejs.org/)

3. **Angular CLI**  
   Instala el CLI de Angular globalmente usando npm:  
   ```bash
   npm install -g @angular/cli

4. **.SQL**  Instalar SQL Server (o SQL Express) localmente o accesible, y asegúrate de que puedas crear bases de datos, usuarios, etc.
5. **Code**   Ten un editor de código como Visual Studio Code o Visual Studio según tu preferencia.
